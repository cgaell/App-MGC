# App MGC

Starter workspace for a React web app built with Vite and TypeScript.

## Available scripts

- `npm run dev` starts the local development server.
- `npm run build` creates a production build.
- `npm run preview` serves the build locally.
- `npm run lint` runs ESLint.
- `npm run format` formats the codebase with Prettier.
- `npm run check` runs lint and build together.

## Structure

- `src/main.tsx` boots the app.
- `src/App.tsx` contains the starter interface.
- `src/styles.css` defines the visual system and layout.

## Next step

Install dependencies and start the app with:

```bash
npm install
npm run dev
```

## Logo

Replace the file at `public/mgc.jpeg` to change the logo shown in the navbar.
